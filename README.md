# Word 图表编号整理助手

这是一个可直接部署到 Cloudflare Pages 的静态网页工具，用来批量整理 `.docx` 文档里混乱的图表编号。

## 功能

- 上传 `.docx` 文件并在浏览器本地处理
- 自定义图表标注规则，支持正则片段与占位符组合
- 支持 `{n}` 递增编号，以及通用的 `{t数字}` 标题编号，例如 `{t1}`、`{t2}`、`{t3}`
- 每个占位符都可单独设置起始编号，默认都是 `1`
- 先重排真正的图表标题，再同步更新正文中的引用标注
- 结果区会高亮每条规则命中的原文片段，便于人工复核
- 支持中文 / English 模式

## 规则写法

- 只有大括号占位符会被改写：
  - `{n}`：真正递增的编号
  - `{t数字}`：按对应级别标题顺序自动编号，例如 `{t1}`、`{t2}`、`{t3}`
- 其它部分按正则匹配，但不会被改动

示例：

- `图{t1}-{n}`
- `表{t1}-{n}`
- `附图\({t1}\)-{n}`
- `Fig\.{t1}-{n}`

说明：

- `图{t1}-{n}` 会把章号部分改成当前一级标题编号，并在每个一级标题范围内让 `{n}` 从起始值重新递增
- 如果规则里用了 `{t3}`，则会按当前三级标题编号替换该位置
- 若某个标题前没有可用的对应级别标题上下文，该占位符位置会保留原数字，并在结果里给出提示

## 使用方式

1. 打开网页并上传 `.docx`
2. 设置规则与各占位符起始编号
3. 点击“扫描并预览”
4. 确认标题和正文引用映射结果
5. 下载修正后的 DOCX

## 部署到 Cloudflare Pages

这是纯静态站点，不需要构建。

### 方式一：Cloudflare Pages

1. 将项目目录上传到 Git 仓库
2. 在 Cloudflare Pages 中连接该仓库
3. 构建设置保持为空：
   - Build command：留空
   - Build output directory：`.`
4. 完成部署

### 方式二：Wrangler

```bash
wrangler pages deploy .
```

## 当前限制

- 仅支持 `.docx`
- `{t数字}` 按标题出现顺序编号，不直接读取标题文字里的现成数字
- 复杂排版文档建议先用结果区人工复核一遍再导出
